Adds a column to the post-listing admin page displaying the post's set publish time.

== Description ==

Say you have multiple posts pending for tomorrow, and want to compare their publish times to make sure some are going out in the morning, and some around dinner.  Using WordPress's standard post-listing tools this can be a pain in the butt.

Here's where postTime comes to the rescue!  Once activated you get to see all your scheduled posts complete with the publishing time - without having to open the quick editor for each and every post.

Official plugin site: http://www.jtb-productions.com/project/posttime/
== Installation ==

1. Upload `postTime.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==
